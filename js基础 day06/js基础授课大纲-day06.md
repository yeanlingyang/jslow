# 对象

## 点语法和中括号语法

1. 点语法   obj.xxx
2. [ ]语法   可以把对象当成数组来看待, 在中括号内支持字符串或者变量



 

## 点语法和中括号语法的区别

1. 点语法: 简洁明了, 但是不支持变量

2. [ ]语法:  支持变量和字符串, 语法更灵活





## 中括号语法的使用场景

点语法完成不了了，就是用[ ]语法

需求: 给对象添加属性

```js
{ }
{ n1: 1, n2: 2, n3: 3, n4: 4 ...... n10: 10 }
```
